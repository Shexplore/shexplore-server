# Shexplore Server

## Setup:
    npm install
`settings/db.json`
    {
      "client": "pg",
      "connection": {
        "host"     : "HOST",
        "user"     : "USER",
        "password" : "PASSWORD",
        "database" : "DATABASE",
        "charset"  : "utf8"
      }
    }


## Running:
    npm start
